1. Estruturas de Deciso

  • De acordo com [MDN Web Docs], "em qualquer linguagem de programao, o cdigo precisa tomar decises e realizar aes de acordo, dependendo de diferentes entradas".
  • Para aprofundamento consulte MDN Web Docs.

1.1. Estruturas de Deciso Simples

  • Sintaxe:
    • A palavra reservada if seguida de um par de parnteses ( ), onde a condio dever ser indicada, e as linhas de cdigo sero inseridas entre as chaves { }.
if (condio) {
    ... linhas de cdigo finalizadas com ponto e vrgula;
}

1.2. Estruturas de Deciso Composta

  • Sintaxe:
    • A palavra reservada if seguida de um par de parnteses ( ), onde a condio dever ser indicada, e as linhas de cdigo sero inseridas entre as chaves { }.
    • E a palavra reservada else seguida de chaves { }, onde sero informadas as demais linhas de cdigo, caso seja falso o teste da condio.
if (condio) {
    ... linhas de cdigo finalizadas com ponto e vrgula;
} else {
     ... linhas de cdigo finalizadas com ponto e vrgula;
}

1.3. Estruturas de Deciso Encadeada

  • Sintaxe: 
if (condio1) {
    ... sero executadas as linhas de cdigo caso condio1 seja verdadeira;
} else if (condio2) {
     ... sero executadas as linhas de cdigo caso condio1 falsa e a condio2 seja verdadeira;
} else if (condio3) {
     ... sero executadas as linhas de cdigo caso condio1, condio2 falsas e a condio3 seja verdadeira;
} else if (condio4) {
     ... sero executadas as linhas de cdigo caso condio1, condio2, condio3 falsas e a condio4 seja verdadeira;
} else {
     ... sero executadas as linhas de cdigo caso todas as condies anteriores foram falsas;
}
  • Exemplo: 
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Estruturas de Deciso</title>
</head>
<body>
    <script>
      let d = new Date();
      let h = d.getHours();
      if (h >= 0 && h <= 12) {
        console.log("Bom dia!!!");
      } else if (h <= 18) {
        console.log("Bom tarde!!!");
      } else if (h > 18) {
        console.log("Boa noite!!");
      }
    </script>
</body>
</html>

1.2 Switch Case

  • De acordo com MDN Web Docs, "As instrues if ... else possuem suas desvantagens. Elas so boas principalmente para casos em que voc tem algumas opes, e cada uma requer uma quantidade razovel de cdigo para ser executado, e / ou as condies so complexas (por exemplo, vrios operadores lgicos)".
  • Ainda segundo [MDN Web Docs], "nos casos em que voc deseja definir uma varivel para uma determinada opo de valor ou imprimir uma determinada instruo dependendo de uma condio, a sintaxe pode ser um pouco incmoda, especialmente se voc tiver um grande nmero de opes".
  • Sintaxe: 
switch (expresso) {  
    case rotulo1: 
        .... declaraes;
        break;  
    case rotulo2: 
        .... declaraes;
        break;
    case rotulo3: 
        .... declaraes;
        break;

    default: 
        .... declaraes;
        break;
}
  • A palavra reserva switch utiliza uma nica expresso / valor como uma entrada e, em seguida, examinam vrias opes at encontrarem um que corresponda a esse valor, executando o cdigo correspondente que o acompanha.
  • A palavra reservada switch seguida de um par de parnteses ( ), onde a expresso dever ser indicada, e as chaves { }, que contero cada uma das escolhas, por meio do uso da palavra reservada case seguida da escolha rtulo :, onde sero informadas as demais linhas de cdigo, caso seja verdadeiro a escolha da expresso.
  • O break; serve para interromper o bloco e seguir normalmente nas linhas de cdigo fora do switch.
  • Exemplo: 
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Estruturas de Deciso</title>
</head>
<body>
    <script>
       tipoFruta = prompt("Indique um tipo de fruta: ");
       switch (tipoFruta) {
         case "laranja":
           console.log("O preo da " + tipoFruta + "  R$ 5.00.");
           break;
         case "P\pera":
           console.log("O preo da " + tipoFruta + "  R$ 5.00.");
           break;
         case "abacaxi":
           console.log("O preo da " + tipoFruta + "  R$ 5.00.");
           break;
         case "abacate":
           console.log("O preo da " + tipoFruta + "  R$ 5.00.");
           break;
          default:
            console.log("No temos esta fruta!!");
            break;
       }
       console.log("Gostaria de mais alguma coisa?");
    </script>
</body>
</html>

1.3. Operador Ternrio

  • O operador ternrio testa uma condio e retorna um valor / expresso se for verdadeira e outro caso seja falso. Muito til no caso de pouco cdigo a escrever.
  • Sintaxe: 
    (condio) ? resposta ao teste verdadeiro :  resposta ao teste falso;
  • Exemplo: 
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Estruturas de Deciso</title>
</head>
<body>
    <script>
            let tipoFruta = prompt("Indique o tipo de fruta: ");
            let resultado = tipoFruta == "Abacaxi" ? "Acertou o tipo de fruta!" : "Errou o tipo, tente novamente!";
            console.log(resultado + "!!");
    </script>
</body>
</html>

2. Estruturas de Repetio

  • For...

  • For ... in ...

2.1. For

  • Um lao for repetido at que a condio especificada seja falsa.
  • O lao for no JavaScript similar ao Java e C.
  • Uma declarao for feita da seguinte maneira: 
for (declarao; condio; passos) {
  ... declaraes
}
  • Exemplo:
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>JS_FOR</title>
</head>
<body>
    <form name="selectForm">
      <p>
        <label for="tipoMusica">Escolha alguns tipos de msica, em seguida, clique no boto abaixo:</label>
        <select id="tipoMusica" name="tipoMusica" multiple="multiple">
          <option selected="selected">R&B</option>
          <option>Jazz</option>
          <option>Blues</option>
          <option>New Age</option>
          <option>Classico</option>
          <option>pera</option>
        </select>
      </p>
      <p><input id="btn" type="button" value="Quantos foram selecionados?" /></p>
    </form>

    <script>

        function howMany(selectObject) {
          var numeroSelecionadas = 0;
          for (var i = 0; i < selectObject.options.length; i++) {
            if (selectObject.options[i].selected) {
              numeroSelecionadas++;
            }
          }
          return numeroSelecionadas;
        }

        var btn = document.getElementById("btn");
        btn.addEventListener("click", function(){
          alert('Total de opes selecionadas: ' + howMany(document.selectForm.tipoMusica))
        });
    </script>
</body>
</html>

2.2. Do ... While

  • Um lao Do ... While repetido at que a condio especificada seja falsa.
  • Executa pelo menos uma vez, antes de testar a condio. 
do {
  ... declaraes;
} while (condio);
  • Exemplo: 
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>JS_DO-WHILE</title>
</head>
<body>
    <script>
        i = 0;
        do {
          i += 1;
          console.log(i);
        } while (i < 5);
    </script>
</body>
</html>

2.3. While

  • Uma declarao While executa suas instrues, desde que uma condio especificada seja avaliada como verdadeira.
  • Se a condio se tornar falsa, a declarao dentro do lao para a execuo e o controle passado para a instruo aps o lao.
  • O teste da condio ocorre antes que o lao seja executado:
    a) se a condio for verdadeira o lao executar e testar a condio novamente.
    b) se a condio for falsa o lao termina e passa o controle para as instrues aps o lao. 
while (condio) {
  ... declaraes;
}
  • Exemplo: 
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>JS_WHILE</title>
</head>
<body>
    <script>
        n = 0;
        x = 0;
        while (n < 3) {
          n++;
          x += n;
        }
        console.log(n);
        console.log(x);
    </script>
</body>
</html>

2.7. For ... In

  • A declarao for ... in executa iteraes a partir de uma varivel especfica, percorrendo todas as propriedades de um objeto ou array.
  • Sintaxe: 
for (variavel in objeto) {
    ... declaraes 
}
  • Exemplo: 
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>JS_FOR-IN</title>
</head>
<body>
    <script>
        /*Ento executar uma iterao para cada elemento e retornar uma lista de string, que ir conter o nome da propriedade e seu valor. */
        function dump_props(obj, obj_name) {
            var result = "";
            for (var i in obj) {
              result += obj_name + "[" + i + "] = " + obj[i] + "\n";
            }
            result += "****";
            return result;
          }

         console.log(dump_props([1, 2, 3, 4], 'item'));
    </script>
</body>
</html>

2.8. For ... Of

  • A declarao for ... of cria uma lao com objetos interativos (incluindo, array, map, set, assim por conseguinte), executando uma iterao para o valor de cada propriedade distinta.
  • Sintaxe: 
for (variavel of objeto) {
    ... declaraes 
}
  • Exemplo: 
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>JS_FOR-OF</title>
</head>
<body>
    <script>
        /* O exemplo a seguir mostra a diferena entre o for...of e o for...in. 
        Enquanto o for...in interage com o nome das propriedades, o for...of interage com o valor das propriedades. */
        let arr = [3, 5, 7];
        arr.foo = "hello";

        for (let i in arr) {
           console.log(i); // logs "0", "1", "2", "foo"
        }

        for (let i of arr) {
           console.log(i); // logs "3", "5", "7"
        }
    </script>
</body>
</html>

Exerccios

1. Elabore um programa que exiba um caracter em x quantidade de linhas. Aumentando a repetio conforme a linha.

x = prompt("Indique o caracter: ");
y = prompt("Indique a quantidade de linhas: ");
exibirCaracter(x, y);

function exibirCaracter(texto, linhas) {
    for(let l=1; l <= linhas; l++) {
        let desenho = '';
        for(let m=0; m < l; m++) {
            desenho += texto;
        }
        console.log(desenho);
    }

}
